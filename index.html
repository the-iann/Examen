Resumen sobre BORDER-RADIUS

BORDER-RADIUS es una propiedad CSS que permite redondear los bordes de un elemento HTML. Es útil para crear elementos con esquinas redondeadas y proporcionar una apariencia más suave y moderna. Al aplicar BORDER-RADIUS, puedes dar forma a los elementos con bordes curvos , ovalados o incluso circulares. Esta propiedad también puede ser combinada con otras propiedades de estilo CSS, como el color de fondo o la sombra, para crear efectos visuales más atractivos. Sin embargo, BORDER-RADIUS no es compatible con todos los navegadores antiguos, por lo que es importante considerar la compatibilidad del navegador al usar esta propiedad. En resumen, BORDER-RADIUS es una herramienta poderosa para agregar estilo y personalidad a tus diseños web.

Comandos de Git

  • git init: Inicializa un nuevo repositorio Git.
  • git add: Agrega cambios al área de preparación.
  • git commit: Guarda los cambios en el repositorio.
  • git status: Muestra el estado de los archivos en el directorio de trabajo y en el área de preparación.
  • git push: Sube los cambios locales al repositorio remoto.
  • git pull: Obtiene cambios del repositorio remoto y los fusiona en el repositorio local.
  • git clone: Clona un repositorio remoto en un nuevo directorio local.
  • git branch: Lista, crea o elimina ramas.
  • git checkout: Cambia entre ramas o restaura archivos.
  • git merge: Fusiona una rama con otra.
